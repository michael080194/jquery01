使用說明
a[target]：選擇 所有元素 a 裡面有 target 屬性
a[target=_blank]：選擇 所有元素 a 屬性 target 值完全等於 _blank
[title~=flower]：選擇 屬性 title 值 包含 flower 字串
[class|=top]：選擇 屬性 class 值 為 top 或開頭字串為 top-
[class^='top']：選擇 屬性 class 值 為 top 開頭的元素
[class$='top']：選擇 屬性 class 值  結尾為 test 的元素
[title*='er']：選擇 屬性 title 值 包含 er 字串
input[type='text']：選擇 元素input , 屬性 type 值 等於 text , 改變該元素的 style
[class='bt1']：選擇 屬性 class 值 等於 bt1 , 改變該元素的 style
