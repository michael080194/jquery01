jquery 物件使用說明
當使用 console.log 顯示 jquery 物件 [object Object]時,會顯示出該物件的 function 內容
要取用物件裡面的值須使用 jquery的方法或屬性;如 alert(temp1.text())
html 物件使用說明
當使用 console.log 顯示 html 物件 [object HTMLDivElement]時,會顯示出該元素的 內容(類似function)
要取用物件裡面的值須使用 html的方法或屬性;如 alert(temp1.get(0).innerText)
其實他是抓 javascript 原生物件
.each 使用說明
當使用 $.each(...) 抓出來的物件是 [object HTMLDivElement]
要抓取物件內容 alert(obj.innerText)
所以若要使用 jquery 的方法 須轉換成 alert($(obj).text());
for 使用說明
當使用 for(x in obj){...} 可顯出該物件可使用的所有 method 和 property
當使用 $.each(obj, function(i, obj1){...}) 可顯出該物件所有 method 和 property的值
